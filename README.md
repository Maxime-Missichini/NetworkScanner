# NetworkScanner
Network Scanner script using Python
## Architecture
- basicScan - Uses only basic python commands
- scapyScan - Uses Scapy (**NEED SUDO**)
---
## Prerequisites
### basicScan
    pip install tqdm
### scapyScan
    pip install scapy
---
## How to run ?
**If in the correct directory**
### basicScan
    python3 basicScan.py
### scapyScan
    sudo python3 scapyScan.py
